1. 主菜单界面（GameMenu）
菜单集成六大功能按钮：开始游戏、退出程序、参数设置、排行榜、成就系统、角色皮肤选择；
 
- 退出：一键关闭整个游戏程序；
- 设置面板：自定义配置游戏初始精力值、对局倒计时时长，自定义开局参数；
- 排行榜/成就：可跳转对应数据界面，支持返回菜单或直接开启新对局；
- 皮肤选择：预选可用角色外观，选中配置本地持久保存。
 
2. 游戏对局界面（GameView）
承载游戏全部核心运行逻辑，包含场景滚动、计时器、碰撞检测、计分、角色状态刷新、道具随机生成等功能，对局结束自动跳转回主菜单。
 
三、核心游戏玩法规则
 
（一）角色操控规则（小球/玩家角色）
 
1. 键盘左右方向键控制角色水平位移，触碰平台自动向上弹跳，上升阶段可穿透上方平台；无平台支撑时角色做自由落体加速下落；
2. 玩家拥有初始精力（生命值），掉落至屏幕底部ddl死线区域、踩踏尖刺平台均扣除1点生命值，生命值清零直接结束对局；剩余生命值则在上次落脚存档平台复活；
3. 角色搭载动态表情系统：常态、紧张、力竭三种神态根据小球状态自动切换 。
 
（二）四类功能平台（Platform）
 
1. 普通平台：稳定落脚、常规弹跳，无伤害、永久保留；
2. 高弹平台：双倍弹跳加成，实现超高跳跃；
3. 易碎平台：单次踩踏后直接破碎消失，仅可借力一次；
4. 尖刺平台：踩踏扣除生命值，增加闯关容错难度。
 
（三）场景滚动生成机制
 
当角色攀升至屏幕竖直一定高度时，全局场景向下平移，平台、金币道具同步下移；移出屏幕范围的资源自动销毁，屏幕上半区实时随机生成全新平台与道具，实现无限向上闯关效果。
 
（四）限时DDL倒计时机制
 
对局开启后启动全局倒计时，倒计时归零或生命值耗尽即游戏结束；倒计时剩余最后5秒启用红色醒目文字弹窗提醒；根据失败原因弹出差异化文案，解锁不同成就
 
四、道具与计分系统（Coin灵感道具）
 
摒弃传统金币计分，改用灵感值作为游戏积分，地图随机刷新四种主题道具：
 
1. GPT道具、高数书：拾取后提升灵感分值；
2. 邮件道具：大量增加灵感值，额外延长对局DDL倒计时；
3. 咖啡道具：恢复角色生命值，触发短时磁吸效果，自动吸附周边所有道具，灵感爆发加成。
额外预留加倍奖励金币拓展位，可后续迭代高分倍率道具。
 
五、特色拓展功能
 
1. 角色皮肤系统
 除了默认小球外，内置小猫、小狗、熊猫三套独立角色外观，支持主菜单手动自选更换，皮肤配置本地文件存储，重启游戏自动沿用上次选择。
 
2. 排行榜：自动存储历史对局灵感分数，留存TOP10榜单数据，菜单排行榜界面可查看；
