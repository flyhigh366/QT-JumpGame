一、操作规则
1.先进入gamemenu菜单界面gamemenu：
开始和结束按钮、排行榜，有三个选项。
鼠标点击操作。 
点击开始则进入gameview游戏主界面，点击结束则游戏结束                        //点击排行榜进入排行榜,排行榜功能均未实现
2.如果进入gameview游戏界面
   开始游戏；
   计时器（30秒）为0或死亡，则游戏结束，返回gamemenu。
二.小球；
 (1)键盘左右方向键控制小球水平移动
 (2)小球接触平台后弹跳向上（小球向上时可以无视上方木板）
 (3)小球不接触平台时匀加速下落  ，落下时落在第一个接触到的platform上。
（4）小球初始3点生命值。
    掉到死线下方，生命值-1；还有生命则回到最近存档平台复活，无生命则游戏结束。 
    踩到带刺木板：生命值-1，无生命直接结束游戏。
三、死线规则
1.屏幕下方y=0处设有死线。
2.小球落至死线以下，生命值-1，无生命直接结束游戏。
四、platform设定
1.普通板块：正常弹跳，无伤害、不消失。 
2.带刺板块：踩踏扣1点生命。
五，限时通关规则
游戏为有限时间闯关，并非无限向上跳跃，30秒。
六.滚动规则
小球到达距屏幕1/2位置时，屏幕开始滚动：金币木板和小球都向下移动，移动长度为屏幕高度的1/2。
不在屏幕内的木板和金币删除。屏幕上方1/2范围内新生成platform和coin。
